# python-task-automation
Python script to automate file organization

## Description
Automates repetitive file management tasks such as organizing and renaming files using Python.

## Problem
Manual file handling is slow and error-prone.

## Solution
A Python script that scans folders, categorizes files by type, renames them using timestamps, and logs all operations.

## Technologies
- Python
- OS Module
- Shutil
- Logging

## How to Run
```bash
python src/main.py
